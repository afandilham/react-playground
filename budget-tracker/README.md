
## Run Locally

Clone the project

```bash
  git clone https://github.com/afandilham/coverpost.git
```

Go to the project directory

```bash
  cd coverpost
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
